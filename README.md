# PreactPress90

90年代レトロデザインの静的ブログジェネレーター

## 特徴

- 📝 Markdownで記事を書ける
- 🎨 90年代風のレトロなデザイン
- ⚡ Preact + Viteで高速ビルド
- 📊 タグの使用状況をバーグラフで表示

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build
```

## 記事の書き方

`posts`フォルダに`.md`ファイルを作成：

```markdown
---
title: "記事タイトル"
date: "2025-01-21"
author: "著者名"
tags: ["Preact", "Vite"]
---

記事の内容をMarkdownで書く
```
